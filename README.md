# Adaptive Franchise Training with Reinforcement Learning

## Overview
This project implements a Reinforcement Learning (RL) based adaptive training system for franchise businesses. The system aims to optimize training programs for franchise employees by adapting to their performance and learning pace.

## Features
- **Personalized Training Modules**: Tailors training content based on individual progress and skill level.
- **Performance Tracking**: Monitors employee performance to adjust training accordingly.
- **User-Friendly Interface**: An intuitive UI for both trainers and trainees.
- **Analytics Dashboard**: Provides insights into training effectiveness and employee performance.

## Getting Started

### Prerequisites
- Python 3.x
- Required Libraries: `numpy`, `pandas`, `tensorflow`, `matplotlib`

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Sripravallika1/adaptive-franchise-training-rl.git
   cd adaptive-franchise-training-rl
   ```
2. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

### Usage
1. Run the training algorithm:
   ```bash
   python train.py
   ```
2. Access the user interface:
   ```bash
   python app.py
   ```
