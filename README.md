# Adaptive Franchise Training

This repository contains the complete adaptive training system with all modules necessary for effective reinforcement learning in franchise training.

## Modules

- **simulator**: Contains the core training simulations and environment setup.
- **metrics**: Contains various metrics for evaluating performance and cognitive load.
- **experiments**: Contains scripts to run different training simulations.

## Setup

To set up the environment, install the required packages:

```bash
pip install -r requirements.txt
```

## Usage

To run the simulation, use the following command:

```bash
python experiments/run_simulation.py
```
